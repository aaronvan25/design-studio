# User Taste Profile — Ranked Favorites & Preferences

## The Core Principle

Animation is NOT the centerpiece. The PROJECT is the centerpiece. Animation SUPPLEMENTS the project.

Three roles animation plays:
1. **Embodying content** (Arboretum) — fractal trees grow because the page IS about growth
2. **Supplementing feelings** (Drift) — gentle particles evoke exotic calm for a perfume brand
3. **Urgency/functionality** (Cipherdeck) — blinking indicators signal the system is active

Excessive movement drowns out meaning. The trick is restraint.

## Technique Preferences (as TOOLS, not goals)

**Loves when grounded in purpose:**
- Flow fields / particle trails
- Halftone / ordered dithering on live content
- L-system fractal growth
- ASCII text rendering
- Dot-hero stippled numbers (gold/copper on dark)
- Spring-animated cards and timeline nodes (Framer Motion)
- Breathing status indicators
- Live canvas waveforms
- Mix-blend-mode color blobs
- Interactive functionality (dropdowns, sorting, filtering)
- Clip-path stencil reveals
- Scrolling ticker tapes

**Hates:**
- Animation for animation's sake
- Too many competing animations
- Inconsistent typography within a page
- Unreadable labels
- Content cut off by viewport
- Rounded corners, gradient buttons, emoji, drop shadows
- Light-mode treatments breaking dark themes
- AI slop — generic component-library aesthetics
